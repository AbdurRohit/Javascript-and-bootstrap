Chapter 1: Introduction to JavaScript

JavaScript is a high-level, interpreted programming language that allows you to add interactivity and dynamic content to web pages. 

// Simple JavaScript code example
console.log("Hello, JavaScript!");

Chapter 2: Variables and Data Types

Variables are used to store data in JavaScript. JavaScript has several data types, including numbers, strings, booleans, and more. 

// Variable declaration and initialization
let name = "John";
let age = 30;
let isStudent = true;

Chapter 3: Functions

Functions are blocks of code that perform a specific task. They can take parameters and return values. 

// Simple function
function add(a, b) {
    return a + b;
}

// Function call
let result = add(5, 10);
console.log(result); // Output: 15

Chapter 4: Conditional Statements

Conditional statements allow you to make decisions in your code based on certain conditions. 

// Simple conditional statement
let num = 10;
if (num > 0) {
    console.log("Positive number");
} else if (num < 0) {
    console.log("Negative number");
} else {
    console.log("Zero");
}

Chapter 5: Loops

Loops are used to repeat a block of code until a certain condition is met. 

// Simple for loop
for (let i = 0; i < 5; i++) {
    console.log(i);
}

Chapter 6: Arrays

Arrays are used to store multiple values in a single variable. 

// Simple array
let fruits = ["apple", "banana", "orange"];
console.log(fruits[0]); // Output: "apple"
console.log(fruits.length); // Output: 3
